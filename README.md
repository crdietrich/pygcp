# Python Based Data Science with Google Cloud Platform  
A collection of methods to simplify and speed up common analytical work on GCP with an emphasis on integration with Pandas workflows.  

For complete Google product API support use the official packages:  
https://github.com/googleapis/google-cloud-python  

## Getting Started  
1. Refer to the `examples` directory and method docstrings for usage.  
2. Add service account, project ID and API key information to `config.ipynb` to create shared Jupyter variables.
3. Run specific service example notebooks.
